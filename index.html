Function Reference

Prime & Number Theory

isprime(n)

1 if n is prime, else 0

prime(n)

Nth prime: prime(1)=2, prime(2)=3...

primepi(x)

Prime counting π(x) — primes ≤ x

nextprime(n) / prevprime(n)

Adjacent primes

primorial(n) / primegap(n)

Product of primes ≤ n / Gap to next prime

totient(n)

Euler's totient φ(n)

mobius(n)

Möbius function μ(n)

sigma(n) / sigma(n,k)

Sum of divisors σ(n) / σ_k(n)

numdiv(n)

Number of divisors d(n)

liouville(n) / mangoldt(n)

Liouville λ(n) / Von Mangoldt Λ(n)

omega(n) / bigomega(n)

Distinct / total prime factors

mertens(n)

Mertens M(n) = Σ μ(k)

radical(n)

Product of distinct prime factors

collatz(n)

Steps to reach 1 in Collatz sequence

digitsum(n) / digitalroot(n)

Sum of digits / Digital root

isperfect(n) / isabundant(n)

Perfect/abundant number tests

Calculus

deriv(expr) or diff(expr)

Numerical derivative d/dx. Example: deriv(x^3) graphs 3x²

diff2(expr) or deriv2(expr)

Second derivative d²/dx²

integral(expr, a, b)

Definite integral ∫ₐᵇ. Use x as upper bound to graph: integral(sin(t), 0, x)

sum(expr, var, start, end)

Summation Σ. Example: sum(1/n^2, n, 1, x)

prod(expr, var, start, end)

Product Π. Example: prod(n, n, 1, x) = x!
